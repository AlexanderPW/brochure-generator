# Brochure Generator

Fun utility to generate brochures from Websites.

Overview
- The primary script is `brochure_generator/brochure_generator.py`.
- Designed to be used as a small CLI demo or imported as a library by other code.

Quick start

1. (Optional) Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies (if present):

```bash
pip install -r requirements.txt
```

3. Run the demo script:

```bash
python brochure_generator/brochure_generator.py
```

Importing

```python
from brochure_generator import brochure_generator
# inspect available functions
help(brochure_generator)
# call generator functions (e.g. brochure_generator.generate(...))
```
